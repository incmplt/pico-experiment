# Raspberry Pi Pico : Experiment Codes

MicroPython Simple Source Code.

* led.py : RasPi Pico 本体のLEDで Lチカ
* get-sensor.py : RasPi Pico の温度センサーの情報を読み出し
* i2c-scan.py : I2C A0,1 に接続のデバイスをスキャンして情報を表示
